# Capstone-Project
A terminal-based Connect 4 game built in Python for the Sigma Labs pre-course capstone project.
Two players take turns dropping coloured pieces into a 6x7 grid and the first to get four in a row (horizontally, vertically, or diagonally) wins.
Play locally against a friend or against the computer.

## How to Run
You'll need Python 3 installed.
1. Clone the repository:
   `https://github.com/masonbudd/Capstone-Project.git`
2. Move into the project folder:
   `cd capstone-project`
3. Run the game:
   `python3 connect_4.py

## How to Play:
- Choose 1-player mode to play against the computer or 2-player mode to play against a friend.
- Player 1 is red, player 2 is yellow.
- On your turn, enter a column number from 1 to 7 to drop your piece.
- Pieces fall to the lowest empty row in that column.
- First player to get four in a row wins.
